\begin{frame}[t]{Operaciones con Variables}\vspace{10pt}

Ejecuta el algoritmo.
\begin{block}{Pulsa:}
	CTRL + B
\end{block}

Las operaciones básicas...
\begin{block}{Matemática básica...}
	y = (2*(x + 2)**(3*x))/5
\end{block}

Que equivale a...

\begin{equation}
	y = \frac{2 \left(x  + 2 \right)^{3x}}{5}
\end{equation}
