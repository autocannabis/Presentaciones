\begin{frame}[fragile]{for - while}\vspace{10pt}

Las funciones c\'iclicas se usan de la siguiente manera:

\begin{center}
\begin{lstlisting}
	for (inicio; fin; incremento) {
		...	
	}
\end{lstlisting}
\end{center}

\begin{center}
\begin{lstlisting}
	while (condicion) {
		...	
	}
\end{lstlisting}
\end{center}

\end{frame}
