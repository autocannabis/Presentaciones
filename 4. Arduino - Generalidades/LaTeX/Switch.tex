\begin{frame}[fragile]{Switch}\vspace{10pt}

Es una funci\'on propia de \textit{C++} que cumple un prop\'ostio similar al de if - else.

\begin{center}
\begin{lstlisting}
	switch (var) {
		case label1:
			...
			break;	
		case label2:
			...
			break;
		default:
			...
			break;
	}
\end{lstlisting}
\end{center}

\end{frame}
